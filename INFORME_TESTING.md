 ## Introducción
 Este informe documenta el proceso de testing de la app Renü desarrollada en Python + Kivy/KivyMD. El objetivo es evaluar la calidad y confiabilidad de los componentes clave mediante pruebas unitarias, de integración y funcionales (según aplique).

 ## Plan de Pruebas
 Ver `TESTING_PLAN.md` para el detalle de casos, datos de entrada y resultados esperados.

 ## Preparación del Entorno
 - Ejecutar desde la raíz del proyecto.
 - Ejecutar pruebas con:
   - Windows (venv del proyecto): `./venv/Scripts/python -m pytest -v`
   - O bien: `pytest -v`

 ## Ejecución de Pruebas (Evidencias)
